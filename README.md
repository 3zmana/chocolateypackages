## NetSetMan

[![Chocolatey Version](https://img.shields.io/chocolatey/v/netsetman)](https://community.chocolatey.org/packages/netsetman)
[![Chocolatey Downloads](https://img.shields.io/chocolatey/dt/netsetman)](https://community.chocolatey.org/packages/netsetman)

**NetSetMan** is a network settings manager that lets you instantly switch between configuration profiles for different locations — ideal for laptops used across multiple networks (home, office, hotel, etc.).

Each profile can store:
- IP address, subnet mask, default gateway
- DNS servers
- WINS servers
- Default printer
- Computer name
- Proxy settings
- Hosts file entries
- Workgroup / Domain
- And more...

### Installation

```powershell
choco install netsetman
```

### Upgrade

```powershell
choco upgrade netsetman
```
